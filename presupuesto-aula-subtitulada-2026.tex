\sectiontitle{Desglose de la partida tecnológica}

La partida de \textbf{\money{450.000}} no corresponde a pago personal del postulante. Se reserva para gastos externos y rendibles que permitan que el prototipo pase de MVP a piloto institucional:

\begin{itemize}
  \item Certificado o firma de código para reducir advertencias de seguridad del instalador en Windows.
  \item Dominio, relay o servicios cloud acotados para conectar celular y computador durante el piloto.
  \item Empaquetado, pruebas de instalación, soporte técnico documentado y ajustes de estabilidad.
  \item Registro autorizado de transcripciones y preparación de funciones para generar resúmenes, glosarios, fichas o infografías.
\end{itemize}

\textbf{Resguardo administrativo:} estos gastos deben ejecutarse mediante compras, proveedores o mecanismos autorizados por AIEP. Si alguna subpartida no fuese rendible, se debe reasignar a equipamiento, evaluación, producción de guía u otros gastos permitidos.

\sectiontitle{Por qué no gastar todo en equipamiento}

La propuesta no es solo una compra de micrófono. El valor del proyecto depende de integrar tecnología, metodología y evidencia:

\begin{itemize}
  \item \textbf{Audio estable:} sin buena captura, el subtitulado pierde precisión; por eso se prioriza un kit de micrófono y respaldo.
  \item \textbf{Mejora técnica:} el piloto requiere estabilidad, registro autorizado y preparación para generar material complementario.
  \item \textbf{Evidencia pedagógica:} se deben aplicar instrumentos simples para medir utilidad, claridad y participación.
  \item \textbf{Transferencia:} la guía de replicabilidad permite que la innovación no quede limitada a una clase o docente.
\end{itemize}

\sectiontitle{Resguardos administrativos}

\begin{itemize}
  \item Los precios pueden cambiar por stock, despacho, proveedor institucional o facturación.
  \item Se recomienda solicitar cotizaciones formales antes de ejecutar la compra.
  \item El ítem de mejoras tecnológicas, seguridad y despliegue debe ejecutarse conforme a las bases y normas internas de AIEP. Puede materializarse como certificado/firma de código, licencias, servicios cloud, soporte técnico, empaquetado, pruebas o proveedor autorizado; no como pago personal al responsable.
  \item Si alguna subpartida no fuese rendible o autorizada, se deberá redistribuir hacia equipamiento, evaluación, producción de guía u otros gastos permitidos por la institución.
  \item El gasto en socialización se mantiene en \money{50.000}, equivalente al 5\% del presupuesto, por debajo del límite prudente de 20\% para difusión/coffee break.
\end{itemize}
